# Installation

## Download from Fab

1. Purchase **Loading Screen Pro** from the [Fab Marketplace](https://www.fab.com/de/sellers/Deblx)
2. The plugin will be added to your Fab library

## Install in Unreal Engine

1. Open the **Epic Games Launcher**
2. Go to your **Fab Library**
3. Find **Loading Screen Pro**
4. Click **Install to Engine**
5. Select your Unreal Engine version (5.1 – 5.7)
6. Click **Install**

## Enable the Plugin

1. Open your Unreal Engine project
2. Go to **Edit → Plugins**
3. Search for "Loading Screen Pro"
4. Check the box to enable it
5. Restart the editor when prompted

## Verification

After restart, open **Edit → Project Settings → Plugins → Loading Screen Pro**.  
You should see the Loading Screen Pro settings page with the **Quick Setup** button.
