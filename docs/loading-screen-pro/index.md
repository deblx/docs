# Loading Screen Pro

Professional loading screen system for Unreal Engine. Replace the default black screen with a fully animated, cinematic loading experience — configured entirely through a Data Asset, no C++ required.

## What it does

- **Media slideshow** with images and 6 transition effects (Fade, Slide, Zoom In, Fade to Black, Cut)
- **3 progress bar styles** — Bar, Segmented, Stepped
- **4 spinner styles** — Ring, Arc, Bars, Dots
- **Ken Burns effect** — slow cinematic zoom on each slide
- **Background music** with fade-in and automatic stop after loading
- **Hints / tips system** with random or sequential cycling and smooth fade transitions
- **Vignette, letterbox bars, logo overlay, and overlay text**
- **Per-level overrides** — different loading screen per map
- **Blueprint API** — show/hide manually, push custom progress, react to slide changes
- **In-editor preview** — live 1280×720 preview window with animations and music
