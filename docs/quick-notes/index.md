# Quick Notes

A comprehensive TODO management system for Unreal Engine 5 that allows developers to create notes both in the viewport and on Blueprint nodes with centralized tracking.

## Overview

Quick Notes revolutionizes how you track development tasks in UE5 by providing multiple ways to create and manage TODOs directly within your workflow.

## Key Features

- **Viewport Notes** - Place floating sticky notes directly in your 3D viewport
- **Blueprint Node Notes** - Attach TODOs directly to Blueprint nodes
- **Centralized Panel** - Manage all notes from a single dedicated panel
- **Project-Wide Tracking** - Never lose track of tasks across large projects
- **Priority System** - Organize notes by priority and category

## Requirements

- Unreal Engine 5.1 or higher
- Available on Fab Marketplace
