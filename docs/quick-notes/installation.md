# Installation

## Download from Fab

1. Purchase Quick Notes from the [Fab Marketplace](https://www.fab.com/de/sellers/Deblx)
2. The plugin will be added to your Fab library

## Install in Unreal Engine

1. Open the Epic Games Launcher
2. Go to your **Fab Library**
3. Find **Quick Notes**
4. Click **Install to Engine**
5. Select your Unreal Engine version (5.0 - 5.7)
6. Click **Install**

## Enable the Plugin

1. Open your Unreal Engine project
2. Go to **Edit → Plugins**
3. Search for "Quick Notes"
4. Check the box to enable it
5. Restart the editor when prompted

The plugin is now ready to use!

## Verification

After restart, Quick Notes should be available:
- **Tools** menu → Quick Notes (opens overview panel)
- Right-click in Viewport → **Add Quick Note here**
- Right-click on any Blueprint node → **Add Quick Note here**

## First Use

No configuration needed! Quick Notes works immediately:
1. Open any level
2. Right-click in the Viewport
3. Select "Add Quick Note here"
4. Fill in your first note

That's it! You're ready to start organizing your work.
