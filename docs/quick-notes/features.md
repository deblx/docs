# Features

## Viewport Notes

Create floating sticky notes directly in your 3D viewport to mark areas that need work.

## Blueprint Node Notes

Attach TODO notes directly to Blueprint nodes for context-specific reminders.

## Centralized Management Panel

View and manage all notes across your entire project from a single dedicated panel.

## Organization & Filtering

Filter notes by priority, category, or location to focus on what matters most.
