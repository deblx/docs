# Installation

## Download from Fab

1. Purchase Duplicate Asset Detector Pro from the [Fab Marketplace](https://www.fab.com/de/sellers/Deblx)
2. The plugin will be added to your Fab library

## Install in Unreal Engine

1. Open the Epic Games Launcher
2. Go to your **Fab Library**
3. Find **Duplicate Asset Detector Pro**
4. Click **Install to Engine**
5. Select your Unreal Engine version (5.5, 5.6, or 5.7)
6. Click **Install**

## Enable the Plugin

1. Open your Unreal Engine project
2. Go to **Edit → Plugins**
3. Search for "Duplicate Asset Detector"
4. Check the box to enable it
5. Restart the editor when prompted

The plugin is now ready to use!

## Verification

After restart, you should see the Duplicate Asset Detector in:
- **Tools** menu → Duplicate Asset Detector
- Right-click any folder in Content Browser → **Find Duplicates**
