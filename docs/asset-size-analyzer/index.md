# Asset Size Analyzer

Analyze and visualize asset sizes in your Unreal Engine project. Quickly identify large assets and optimize your project's memory footprint.

## Overview

Asset Size Analyzer is a **free** editor tool that provides comprehensive insights into your project's asset sizes. Whether you're optimizing for memory budgets, preparing for shipping, or conducting asset audits, this plugin gives you the data you need to make informed decisions.

## Key Features

### Comprehensive Asset Scanning
- Scan entire project or specific folders instantly
- Automatic detection of all asset types in your project
- Fast, efficient scanning even for large projects

### Smart Filtering & Sorting
- Filter by asset type (Textures, Meshes, Materials, etc.)
- Set size thresholds to focus on large assets
- Sort by any column - size, type, name, path
- Instant search and filtering

### Detailed Size Information
- **Disk size:** Actual file size on disk
- **Memory usage:** Estimated runtime memory footprint
- Complete asset path and type information
- Easy-to-read table format

### Professional CSV Export
- One-click export with full asset information
- Perfect for reports and documentation
- Share with team or stakeholders
- Track size changes over time

### Native UI
- Fast, responsive interface matching UE Editor style
- Sortable columns with visual indicators
- Clean, professional appearance
- Integrated into Editor toolbar and menus

### Zero Setup Required
- Enable plugin and start analyzing immediately
- No configuration needed
- Works out of the box

## Perfect For

- **Optimizing project size** before shipping
- **Identifying oversized** textures and meshes
- **Memory budget management** and tracking
- **Asset audits** and quality reviews
- **Performance optimization** planning

## Technical Details

- **Supported Engine Versions:** 5.0, 5.1, 5.2, 5.3, 5.4, 5.5, 5.6, 5.7
- **Platforms:** Windows, Mac
- **Module Type:** Editor Only (no runtime impact)
- **Network Replicated:** No
- **Price:** FREE

## Supported Asset Types

- Textures (Texture2D, TextureCube, RenderTarget)
- Static Meshes and Skeletal Meshes
- Materials and Material Instances
- Sounds (SoundWave, SoundCue)
- Animations (AnimSequence, AnimMontage, Blendspace)
- Blueprints and C++ Classes
- Niagara Systems and Emitters
- And many more - automatically detects all types
