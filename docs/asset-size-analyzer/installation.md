# Installation

## Download from Fab


1. Go to the [Fab Marketplace](https://www.fab.com/de/sellers/Deblx)
2. Find **Asset Size Analyzer**
3. Click **Add to Cart**
4. Complete checkout
5. The plugin will be added to your Fab library

## Install in Unreal Engine

1. Open the Epic Games Launcher
2. Go to your **Fab Library**
3. Find **Asset Size Analyzer**
4. Click **Install to Engine**
5. Select your Unreal Engine version (5.1, 5.2, 5.3, 5.4, 5.5, 5.6 or 5.7)
6. Click **Install**

## Enable the Plugin

1. Open your Unreal Engine project
2. Go to **Edit → Plugins**
3. Search for "Asset Size Analyzer"
4. Check the box to enable it
5. Restart the editor when prompted

The plugin is now ready to use!

## Verification

After restart, you should find Asset Size Analyzer in:
- **Window** menu → Asset Size Analyzer
- Editor toolbar (optional icon)

## First Use

On first use, the plugin will compile automatically. This is normal and only happens once. After compilation, you can start analyzing your assets immediately!

## Performance Note

Initial scan may take 5-30 seconds for large projects (10,000+ assets). This is normal - subsequent scans and operations are instant.
